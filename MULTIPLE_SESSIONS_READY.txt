🧪 TESTING WORKFLOWS

WORKFLOW 1: Quick Test (2 minutes)
   1. Go to /mock-users
   2. Click "Create Test Users"
   3. Switch to student1
   4. Navigate to Events page
   5. Go back to /mock-users
   6. Switch to student2
   7. Check same Events page
   8. Compare experiences!

WORKFLOW 2: Custom Users (3 minutes)
   1. Go to /signup-mock
   2. Create custom user (e.g., john@student.nu.edu.pk)
   3. Explore the app
   4. Logout
   5. Go to /mock-users
   6. See your new user in the list
   7. Switch between users

WORKFLOW 3: Password Testing (2 minutes)
   1. Create test users
   2. Note: all use password123
   3. Logout
   4. Try signing in with wrong password
   5. Should show error
   6. Sign in with correct password
   7. Should succeed
