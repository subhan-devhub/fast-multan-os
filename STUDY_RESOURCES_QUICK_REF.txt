🔥 FIREBASE STRUCTURE (Ready for Implementation)

FIRESTORE COLLECTION: studyResources/
  ├─ {documentId}
  │    ├─ title: string
  │    ├─ type: "pastpaper" | "quiz" | "assignment"
  │    ├─ subject: string
  │    ├─ semester: number (1-8)
  │    ├─ year: number
  │    ├─ description: string
  │    ├─ fileURL: string
  │    ├─ fileName: string
  │    ├─ fileSize: string
  │    ├─ uploadedBy: string
  │    ├─ uploadedByUID: string
  │    ├─ downloadCount: number
  │    └─ createdAt: timestamp

STORAGE PATH:
study-resources/{type}/{subject}/{filename}

Example:
study-resources/pastpaper/Programming-Fundamentals/pf-mid-2024.pdf
