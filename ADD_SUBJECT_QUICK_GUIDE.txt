🚀 HOW TO ADD A SUBJECT

STEP 1: Click "➕ Add Subject" button (green, in filter bar)

STEP 2: Fill in the modal form
        ┌────────────────────────────────────────┐
        │  Add New Subject                       │
        ├────────────────────────────────────────┤
        │  Subject Name*                         │
        │  [Object Oriented Programming    ]     │
        │                                        │
        │  Semester*                             │
        │  [Semester 2              ▼]          │
        │                                        │
        │  💡 Note: Custom subjects are shared   │
        │                                        │
        │  [Add Subject]  [Cancel]               │
        └────────────────────────────────────────┘

STEP 3: Click "Add Subject" → Subject added instantly!

STEP 4: Subject now appears:
        ✓ In filter dropdown (all semesters)
        ✓ In upload subject dropdown (only its semester)

✅ VALIDATION RULES

Subject Name:
  ✓ Minimum 3 characters
  ✓ Required
  ✗ Cannot duplicate existing subject
  ✓ Case-insensitive check

Semester:
  ✓ Must select 1-8
  ✓ Required

Examples:
  ✓ "Object Oriented Programming" → Valid
  ✓ "OOP" → Valid (3 chars)
  ✗ "OO" → Invalid (too short)
  ✗ "Calculus" → Invalid (already exists)
  ✗ "" → Invalid (empty)
